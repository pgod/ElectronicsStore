# ElectronicsStore

Basic and neat webstore, created with Spring MVC with all configuration created in Java config classes.

Technologies used:
- Spring 4
- Spring MVC
- JSP
- Hibernate 5 (orm mapping)
- Spring Security
- XML & JSON parsing - for REST web service (shopping cart implementation)
- JUnit
- Mockito
- Apache Tiles
- Spring WebFlow
- MySQL
- Joda Time
- JavaMail
- Apache FreeMarker
